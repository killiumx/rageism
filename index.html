Discord Setup Guide
Connect RAGEISM to your Discord server in minutes
Method 1 — Webhook (Easiest, No Coding)
1
Open Your Discord Server
Go to the channel where you want RAGEISM to post — e.g. #match-results or #announcements.
2
Edit Channel → Integrations → Webhooks
Click the gear next to the channel name → Integrations → Webhooks → New Webhook. Name it "RAGEISM Bot" and press Copy Webhook URL.
3
Paste URL in Admin Settings
Go to Admin → Settings, paste the URL in the Discord field and save. The app will automatically POST to Discord when match results are saved.
4
Done! Test It
Log a test result. Your Discord channel should receive a message like: Match M012 — Erangel | Winner: Ghost Squad | Top Fragger: SNIPER (14K)
Webhook payload (auto-sent on result save)
POST {webhook_url} { "username": "RAGEISM", "avatar_url": "https://...", "embeds": [{ "title": "Match Results — M012", "color": 6304479, "fields": [ {"name":"Map","value":"Erangel","inline":true}, {"name":"Winner","value":"Ghost Squad","inline":true}, {"name":"Top Fragger","value":"SNIPER (14K)","inline":true} ] }] }
Method 2 — Discord Bot (Advanced)
1
Create App in Developer Portal
Go to discord.com/developers/applications → New Application. Name it "RAGEISM Bot".
2
Add a Bot + Copy Token
Bot tab → Add Bot. Enable Message Content Intent. Press Reset Token and copy it. Store this securely — never expose it in client-side code.
3
Invite Bot to Your Server
OAuth2 → URL Generator. Scopes: bot + applications.commands. Permissions: Send Messages, Embed Links. Open the URL and add to server.
4
Call Discord API from Supabase Edge Function
In your Supabase Edge Function, send messages via POST /channels/{id}/messages with header Authorization: Bot TOKEN.
Slash Commands to Build
/leaderboardPost top 5 fraggers
/stats [IGN]Show player stat card
/results [matchID]Post match results
/register [IGN]Register from Discord
/golive [matchID]Announce match live
Recommended: discord.js v14 (Node) or discord.py (Python)
Supabase Integration — Quick Start
① Create Project
Go to supabase.com → New Project. Create tables: players, teams, matches, result_logs, polls.
② Get API Keys
Project Settings → API: copy Project URL and anon key. Add both as JS constants at the top of the file.
③ Replace DB calls
Swap DB.get('players') with await supabase.from('players').select('*'). Use .insert(), .update(), .delete() for writes.
Add Supabase JS to this file (in <head>)
<script src="https://cdn.jsdelivr.net/npm/@supabase/supabase-js@2"></script> const supabase = supabase.createClient( 'https://YOUR_PROJECT.supabase.co', 'YOUR_ANON_KEY' ); // Fetch players (replaces DB.get('players')) const { data: players } = await supabase .from('players') .select('*') .order('kills', { ascending: false });
